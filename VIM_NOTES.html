 32 = Yanking (Copying) Text =
 33     - Like deletions and changes, yanking is applied over some range of
 34       characters
 35     - Text is pasted with p (after cursor) or P (before cursor)
 36     - Text is yanked into registers. By default the last yanked thing is
 37       available when executing a "p" or "P" command
 38     - What registers are available? :reg
 39     - Deleting automatically yanks too!
 40 
 41 = Using Explicitly Named Registers =
 42     - "[x]{d|c|y}{motion}
 43     - "[x]{p|P}
 44 
 45 = Registers =
 46     - The unnamed register ". All deletes and yanks go here by default. All
 47       puts use this register if no other is specified
 48     - Numbered registers 0-9.
 49         - 0 contains most recent yank
 50         - 1 contains most recent delete
 51     - Small delete register -
 52     - Last search pattern register /
 53     - [Others]
